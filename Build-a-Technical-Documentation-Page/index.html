About

FireBlaze7 is a new tool for frontend developers wich helps you with the control of the different kind of services and dependencies without a line of code.

It's possible by the nice interface developed by our team, easy to learn and compatible will all kind of projects.

Forget about all the code that is not comfortable for designers and other non-programmers people :)
Installation

For the installation in ubuntu 20.04 LTS, java 8 is required, so:

sudo apt-get install openjdk-8-jdk

And python3

sudo apt-get install python3

Then just install FireBlaze7

sudo apt-get install fireblaze7
Use

After the installation of FireBlaze7 is time to run it from the console, as the use of FireBlaze7 is simple to run it. Just need to type the command:

FireBlaze7

And it's done, enjoy it ;)
